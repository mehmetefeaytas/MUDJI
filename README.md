# Model Uydu Dijital İkiz Simülasyonu (MUDJİ)

Bu proje, Teknofest Model Uydu Yarışması kapsamında
model uydu sisteminin düşüş, ayrılma, paraşüt açılması,
sensör verileri ve telemetri süreçlerini simüle eden
bir **dijital ikiz** çalışmasıdır.

## Özellikler
- Fizik tabanlı düşüş modeli
- Taşıyıcı & görev yükü ayrılması
- Paraşütlü iniş simülasyonu
- Sensör gürültüsü (IMU, GPS, basınç, sıcaklık)
- Telemetri paket üretimi
- Gerçek zamanlı zaman damgası
- Grafiksel analiz
